1. What is Classic Snake?

Classic Snake is a 2D snake inspired game made for mobile. It is available on PlayStore. It was created and is still currently a product from Theana Productions, a French company.

2. What information do we collect?

Classic Snake doesn’t need to collect any information in order to run correctly. The game will, however, automatically connect to your Google Play account in order for you to be linked to the Google leaderboard and advancements of Classic Snake. It will also need an access to your device's storage in order for the game files to be saved. Those game files can be found on most devices here "/storage/emulated/0/Android/data/com.TheanaProductions.ClassicSnake/files/Saves/savedInfos.gtf".
